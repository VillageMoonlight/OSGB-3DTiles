### Bug 修复：几何与纹理回归错误

**问题**：上阶段修改 `TextureProcessor.cpp` 后，转换输出的几何错位、纹理贴图错误（UV 错乱）。

**根因定位**：
- `TextureProcessor.cpp` 修改引入了纹理坐标顺序错误：顶点/索引数组与纹理坐标数组对齐关系被打乱
- `GlbWriter.cpp` 顶点属性 buffer 偏移计算未同步更新

**修复**：
- 恢复 `TextureProcessor.cpp` 进纹理坐标写出顺序，与顶点数组严格对齐
- 修正 `GlbWriter.cpp` 中 accessor 的 `byteOffset` 和 `bufferView` 计算

**验证**：重新转换测试数据，几何正确，纹理贴图无错位。

---

### Bug 修复：KTX2 压缩编译错误

**问题**：启用 KTX2 纹理压缩选项（`compressTexture: true`）时编译报错，ktx 库 API 调用不兼容。

**原因**：ktx 库版本升级后 API 变化（`ktxTexture2_Create` 参数调整），相关调用未同步更新。

**修复**：更新 `TextureProcessor.cpp` 中的 KTX2 API 调用以兼容新版 ktx 库。

---

### Bug 修复：OSGB 文件路径包含中文/特殊字符

**问题**：Windows 8.3 短路径转换失败时，OSG 读入 OSGB 失败，日志显示"文件不存在"。

**修复**：
- 在读取前对输入路径进行 8.3 短路径转换（`GetShortPathNameW`）
- 对短路径转换失败的情况降级处理，使用 UTF-8 原始路径尝试读取

---

### Bug 修复：GUI 日志刷新导致界面冻结

**问题**：转换大量 Block 时，日志输出频繁，GUI `Tk` 主线程被日志 `insert` 操作阻塞，界面无响应。

**修复**：
- 日志消息通过 `queue.Queue` 异步传递，主线程 `after(100, _poll)` 轮询
- 限制单次 poll 最多处理 50 条日志，避免 UI 被长时间独占
